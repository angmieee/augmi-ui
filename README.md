## 开始使用

1.添加 CSS 样式
  
  使用本框架前，请在 CSS 中开启 border-box
  
  ```
  *，*::before,*::after{ box-sizing:border-box; }
  ```
  
  你还需要设置默认颜色等变量（后续会改为 SCSS 变量）
  
  ```
  :root {
      --button-height: 32px;
      --font-size: 14px;
      --button-bg: white;
      --button-active-bg: #eee;
      --border-radius: 4px;
      --color: #333;
      --border-color: #999;
      --border-color-hover: #666;
   }
  
  ```
  
2.安装 augmi
```$xslt
npm i --save augmi

```
